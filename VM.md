# The Vagrant VM

This VM has all the required software to run the lab, including:

* Apache Geode (Incubating)
* Spring XD 2.0 snapshot
* Apache Spark 1.3.1
* Apache Zeppelin (Incubating)

# Instructions

1. Starting the VM
```
$ vagrant up
```
1. Using SSH
```
$ vagrant ssh
[vagrant@stocks-vm, load: 0.01]
~ $
```
1. Stopping the VM
```
$ vagrant halt
```
1. Run the test script
```
$ ?
```
